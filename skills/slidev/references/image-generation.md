# Image Generation Pipeline (SP2)

## 1. Overview

SP2 auto-generates images for three layouts (image-focus / image-text-split / two-columns with image pattern) using Google Gemini 2.5 Flash Image. Runs as part of `build-deck.sh`; not invoked during `run.sh dev`.
